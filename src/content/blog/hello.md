Bu site WordPress'ten Astro'ya taşındı. Birkaç hedef vardı:

- **Hafif olsun.** Sayfa başına neredeyse hiç JavaScript, ölçülebilir derecede
  hızlı yükleme. Statik HTML + minimal CSS.
- **İçerik kontrolü bende olsun.** Yazılar artık `.md` dosyaları olarak
  GitHub'da; veritabanı, plugin bakımı, güvenlik yamaları yok.
- **SEO bozulmasın.** Eski WordPress permalink'leri yeni URL'lere yönlendiriyor;
  sitemap, RSS, structured data hepsi yerinde.

Eski yazılar import scripti çalıştırılınca yerli yerine oturacak.
